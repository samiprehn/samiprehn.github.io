# Sami's Sites 🤖

A personal portfolio landing page linking to my web projects.

Visit: [samiprehn.github.io](https://samiprehn.github.io)

## San Diego

### 🌅 [San Diego Sunset](https://samiprehn.github.io/sd-sunset/)
Tonight's best San Diego sunset viewpoint, ranked by NWS cloud cover plus layered TAF data and a live GOES-18 satellite. Optional ntfy push when any spot grades A+.

### 🌌 [Skywatch SD](https://samiprehn.github.io/skywatch/)
When and where to see stars near San Diego, this week. Combines NWS forecast with sun/moon ephemeris and a Bortle multiplier per site.

### 🏖️ [San Diego Beaches](https://samiprehn.github.io/sd_beaches/)
Weather, ocean conditions, and live webcams for SD beaches.

### 🐕 [San Diego Dog Beaches](https://samiprehn.github.io/sd_dog_beaches/)
Weather, ocean conditions, and live webcams for SD's dog-friendly beaches.

### 🧺 [San Diego Farmers Markets](https://samiprehn.github.io/sd_farmers_markets/)
Which farmers markets are open today across San Diego County.

## Civic

### 🗳️ [Ready Set Vote](https://samiprehn.github.io/ready-set-vote/)
Research your ballot and print a compact guide. Polling location, candidate links, FEC donations, and GovTrack legislative records.

### 🏛️ [My Reps](https://samiprehn.github.io/my_reps/)
See how your two senators and House rep have been voting — committees, sponsored bills, recent votes with majority/minority indicators.

## Tools & utilities

### 🍽️ [Winner Dinner](https://samiprehn.github.io/winner_dinner/)
Group restaurant picker — vote together using ranked-choice voting.

### 🛒 [Cart Karma](https://samiprehn.github.io/cart-karma/)
Ethical alternatives to Amazon — independent stores, co-ops, B Corps, and secondhand marketplaces.

### 📊 [Data Analyzer](https://samiprehn.github.io/data-analyzer/)
Drop in a CSV and get column profiling, hypothesis tests, correlation heatmaps, and ML recommendations — all in your browser.

## Built for friends

### ⚽ [NWSL Fantasy League](https://nwslfan.github.io/)
A 2026 NWSL pick'em league tracker built and maintained for a friend's league. 27 weekly picks rounds backed by a Google Apps Script form, with results auto-derived from the schedule sheet.

### 🛼 [Roll Call](https://mprehn.github.io/roll_call/)
Roller skating spots in San Diego — built for a friend, lives under their GitHub account.

## Built with

- Vanilla HTML, CSS, JavaScript — single-file apps, no build step
- Tailwind CSS via CDN where useful
- Free public APIs: NWS, NOAA, Census Geocoder, GovTrack, FEC, USGS, AviationWeather.gov
- A few Cloudflare Workers when an API needs a key or a CORS proxy
- Built collaboratively with Claude (Anthropic) 🤖
